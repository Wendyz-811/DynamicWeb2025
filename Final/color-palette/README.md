## Overview

This project is a color palette management tool that helps designers and developers quickly generate harmonious color schemes, save their favorites, and organize them efficiently. Whether you're working on a new design project or just exploring color combinations, this tool makes it easy to find and keep track of your perfect palettes.

## Key Features

### **Generate Random Palettes**
- One-click generation of 5 harmonious colors
- Uses the `randomcolor` library for beautiful color combinations
- Each palette is visually stunning and ready to use

### **Save & Organize**
- Save your favorite palettes with custom names
- Add tags for easy categorization (e.g., "professional", "calm", "vibrant")
- All data persists in a local database

### **Manage Your Collection**
- View all saved palettes in one place
- Edit palette names on the fly
- Delete palettes you no longer need
- Clean, intuitive user interface

### **Visual Color Display**
- Each color displays its HEX code
- Easy-to-read format
- Responsive grid layout
